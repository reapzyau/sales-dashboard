# Sales Dashboard

Live sales call and Calendly booking dashboard.

## Features

- Today's upcoming sales calls from Google Calendar
- Recent Calendly bookings from Gmail
- Auto-refresh every 30 seconds
- Mobile-friendly responsive design

## Tech Stack

- Static HTML/CSS/JavaScript frontend
- Netlify Functions (serverless) for API calls
- Google Calendar API
- Gmail API

## Deployment

Deployed via Netlify with automated GitHub integration.
